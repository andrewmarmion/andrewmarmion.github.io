Here is a selection of books/online resources that I think are worthwhile. In no particular order.

General Software Development

  • Dependency Injection Principles, Practices and Patterns by Mark Seemann and Steven Van Deursen
  • Working Effectively with Legacy Code by Michael C. Feathers
  • Kill It with Fire: Manage Aging Computer Systems (and Future Proof Modern Ones) by Marianne Bellotti
  • A Philosophy of Software Design by John Ousterhout
  • Refactoring: Improving the Design of Existing Code, Second Edition by Martin Fowler

Swift and SwiftUI

  • SwiftUI Animations Mastery (iOS 15) by Mark Moeykens
  • SwiftUI Views Mastery (iOS 15) by Mark Moeykens
  • Swift Style: An Opinionated Guide to an Opinionated Language - Second Edition by Erica Sadun
  • iOS Unit Testing by Example: XCTest Tips and Techniques Using Swift by Jon Reid
