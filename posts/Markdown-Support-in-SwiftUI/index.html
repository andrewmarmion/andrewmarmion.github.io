Markdown support in SwiftUI
  • Article
  • Markdown
  • SwiftUI

Published on 18 September 2021

iOS 15 has built markdown support directly into SwiftUI. You can use it quite easily by just writing your markdown string directly into the Text() initializer.

Here are some examples:

struct ContentView: View {

    var body: some View {
        VStack(spacing: 20) {

            Text("This is text without formatting.")
            
            Text("This is **bold** text.")
            
            Text("This is *italic* text.")

            Text("This is ***bold and italic*** text.")

            Text("This is ~~strikedthrough~~ text.")
            
            Text("`This is monospaced text`")
            
            Text("[This is a link that works](https://andrewmarmion.com)")
            
        }
    }
}

However you may not want to use markdown in your text, so how do we opt out of it. Well it is quite easy to do so.

  1. If you have your string as a variable then SwiftUI will use the Text() initializer where you are using an actual String and it will render it without localization and no markdown. You can read more here.

    1. If you use Text(verbatim:) then it will use no localization and no markdown. You can read more here.

    1. If you want to use localization but no markdown then you need to create a localizedString using String(localized:) and then pass that to the Text(). You can read more here.

      struct ContentView: View {

    let stringVariable = "This is not *markdown*"

    let localizedString = String(localized: "This is not *markdown*")

    var body: some View {
        VStack(spacing: 20) {

            // 1
            // No localization no markdown
            // https://developer.apple.com/documentation/swiftui/text/init(_:)-9d1g4
            Text(stringVariable)

            // 2
            // No localization no markdown
            // https://developer.apple.com/documentation/swiftui/text/init(verbatim:)
            Text(verbatim: "This is not *markdown*")

            // 3
            // This is a localized string but no markdown
            // https://developer.apple.com/documentation/swift/string/3867985-init
            Text(localizedString)
        }
    }
}
