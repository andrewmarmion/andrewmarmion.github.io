How to record a screenshot of video of the iOS simulator
  • Article
  • Recording
  • Simulator
  • SwiftUI

Published on 4 September 2021

Recording a screenshot or a video of the iOS simulator used to be a convoluted affair. You would need to use some specialist screen recording software: like Camtasia or QuickTime, or you would have to take a screenshot of your monitor and then crop it.

However these days are gone.

Screenshots

There are several ways to take a screenshot.

Keyboard shortcut

With the simulator selected, all you need to do is press command ⌘ + S and it will take a screenshot.

Snapshot button

If you haven't looked closely at the simulator you may have missed this one. On the top of the window there is a camera. Just click that and it will take a screenshot.

File menu

With the simulator selected you can click File then Save Screen and it will take a screenshot.

Terminal

It is actually possible to take a screenshot from the terminal, making sure that you have the simulator running, open the terminal and type the following:

xcrun simctl io booted screenshot <filename>.<extension>

This would usually be something like this

xcrun simctl io booted screenshot mySnapshot.png


There are actually a few options that you can use with this:

For example if you would rather have jpegs that pngs you can set the type (the default is png):

--type= and choose one of the following bmp, gif, jpeg or png For non-rectangular displays like the iPhone X you can apply a mask using:

--mask= and then choosing black. This will take a screenshot with rounded corners and the notch.

What do I use?

In my day-to-day I use the keyboard short cut or the button on the simulator.



Recording

Recording the screen has come along way from having to use external software. It can now be done in the simulator similarly to the screenshots.

Keyboard shortcut

With the simulator selected, all you need to do is press command ⌘ + R and it will take a screenshot.

Record button

There is a secret record button on the simulator. With the simulator selected, hold down the option and you will notice that the camera button changes at the top of the screen. Click that button to start recording, you can also click that button to stop recording.

File menu

With the simulator selected you can click File then Record Screen and it will record the screen.

You can either click the recording button on the simulator (see the Record button sectioon above), or you can return to the File menu and click Stop Recording

Terminal

Like taking a screenshot you can also record the simulator from the terminal.

xcrun simctl io booted recordVideo <filename>.<extension>

This would usually be something like this

xcrun simctl io booted recordVideo myRecoding.mov

When the recoding stops

When you stop the recording a small thumbnail will appear at the right-hand-side of the simulator. If you right-click on that it will allow you to save the recording as an animated gif.

What do I use?

Like the screenshot, I mostly use the keyboard shortcut or the simulator record button.

Alternatives

One alternative to using the simulator is to use a tool like RocketSim. This is a great little tool by Antoine van der Lee/ and it supercharges the functionality of the simulator. I would recommend checking it out.

I actually use this at work, because it has the capability to show the touches on the screen. This makes sharing videos with stakeholders even easier, as where you tap is clearly highlighted. For this feature alone, it is worth its weight in gold.

I have been a user of this app since the Beta version, and have not received any incentive for writing the above.
